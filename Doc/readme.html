Installation

PlayLiveATC comes in two versions:

  • PLA_<version>_Win64_VLC.zip
    (2.2MB) is a Windows-only version, recommended for Winodws users, which includes all necessary files including VLC. Just unzip the archive and move the entire PlayLiveATC folder under <X-Plane>/Resources/plugins and you are done: Start X-Plane!
  • PLA_<version>_NoVLC.zip
    includes all platforms (Win, Mac, Linux) but no VLC files. You have to install VLC separately.
    For MacOS just install VLC into the standard folder /Applications/VLC.app.
    For all other OS please refer to the installation documentation.
